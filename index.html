An image container that takes up the same space even if the image does not load.

This is to avoid sudden layout change when the images load, and to help it stay consistent when images are missing.

  • src: The image src.
  • width: Force the element's width.
  • ratio: The image ratio. Defaults to 16:9.
  • alt: The image alt text.
